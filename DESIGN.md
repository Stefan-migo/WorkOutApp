# WorkOutApp — Burgundy Monochrome

> Category: Creative & Artistic

> Surface: web

*Interval training app — cero setup, cero fricción. Diseño warm, grounded, monocromo.*

Workout timer y planificador de intervalos. Construí tu sesión en segundos, ejecutala sin distracciones.

## Color Palette

| Role | Name | Hex | Usage |
| --- | --- | --- | --- |
| background | Background | `#f7f3f1` | page canvas — rose off-white |
| surface | Surface | `#ffffff` | cards, modals, raised surfaces |
| foreground | Foreground | `#1e1416` | body text — near-black cálido |
| muted | Muted | `#8a7678` | secondary text, captions |
| border | Border | `#dccccc` | hairlines, dividers |
| accent | Accent | `#7a1a28` | primary CTA, active states, Work intervals |
| accent-secondary | Accent secondary | `#3a7050` | success / completion states |

### Interval colors (burgundy family)

| Name | Hex | Usage |
| --- | --- | --- |
| Prepare | `#9a6a6e` | prepare blocks |
| Work | `#7a1a28` | work blocks (same as accent) |
| Rest | `#b89a9a` | rest blocks |
| CoolDown | `#4a252a` | cooldown blocks |

## Typography
- **Display:** TWK Lausanne — weights 400, 700 — fallbacks: system-ui, -apple-system, Segoe UI, Helvetica Neue, Arial, sans-serif
- **Body:** TWK Lausanne — weights 400, 700 — fallbacks: system-ui, -apple-system, Segoe UI, Helvetica Neue, Arial, sans-serif
- **Timer:** JetBrains Mono, SF Mono, ui-monospace, monospace — huge countdown numerals

## Voice & Tone

- **Adjectives:** Clean, minimal, high-contrast, warm, grounded, functional, no-nonsense
- **Tone:** Direct, instructivo. Like a personal trainer que respeta tu tiempo. UI labels en inglés, contenido en español.

### Messaging pillars
- Zero setup, zero fricción — Sin registro, sin backend, sin app store
- Guided by design — No dead seconds. Every interval is thought out.
- Total control — From a quick tabata to a 12-week program.

### Vocabulary
- **Use:** Workout, interval, cycle, set, rep, Prepare, Work, Rest, CoolDown, Timer, countdown, sequence, Start, pause, skip, restart
- **Avoid:** Training como default (prefer workout/sesión), Gamification buzzwords (achievement, badge), Stock fitness imagery

## Imagery

- **Style:** Functional exercise line-art, clean silhouettes, high-contrast, grayscale
- **Subjects:** Calisthenics, bodyweight, dumbbell, mobility — real postures
- **Treatment:** Solid fills on neutral backgrounds, no decorative illustration
- **Avoid:** Stock photography, cartoons, emoji as exercise icons

## Layout

- **Radius:** 8px
- **Border weight:** 1px
- **Spacing:** 8px baseline grid

### Posture rules
- Single accent (`#7a1a28`) used at most twice per screen
- No gradients, no emoji
- Dark mode default for timer view
- Mobile-first: 360px base, expand to tablet & desktop
- Timer numerals always JetBrains Mono
